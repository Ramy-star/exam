# **App Name**: GitGrind

## Core Features:

- Tabbed Lectures: Tabbed lecture organization, allowing users to switch between different learning modules.
- Content rendering: MCQ questions and written questions based on user data, available within the relevant tabs.
- Show Answers: Reveal answers to multiple-choice and written questions.
- Theme Toggle: Smooth theme toggle between light and dark modes, and persistent user preference using local storage.
- Quiz Builder Tool: Use generative AI to build custom quiz questions based on web page content provided by the user, with intelligent formatting.
- Animation effects: Enable progressive loading of questions and answer with staggered animations when switching lectures or displaying questions and solutions.

## Style Guidelines:

- Primary color: Dark blue (#2563EB) evoking a sense of trust and focus suitable for educational content. Converted from HSL(217, 83%, 53%)
- Background color: Light gray (#F5F7FA) creating a clean and distraction-free environment for learning. Converted from HSL(220, 24%, 95%)
- Accent color: Purple (#4F46E5) for interactive elements. A strong contrasting tone draws attention and increases usability, while still harmonizing well with the overall scheme. Converted from HSL(248, 77%, 59%)
- Body font: 'PT Sans', sans-serif, for clear and accessible reading experience.
- Headline font: 'Coiny', cursive for the main title of the application, to impart some visual character and branding.
- Section title font: 'Calistoga', serif to impart some additional differentiation and improved aesthetics.
- Font Awesome icons to provide visual cues and enhance usability
